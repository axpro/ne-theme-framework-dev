## Requirements

* Node.js
* Bower
* Gulp

## Install build system

Before we can build the NextEuropa theme framework we need to install the build system
itself. This can be done using Node.js and npm:

```
$ npm install -g gulp bower
$ npm install
$ bower install
```

## Building the theme

```
$ gulp
```
