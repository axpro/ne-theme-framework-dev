Meta

Meta is a component that is used for displaying meta information in single pages and listings. It could also be reused in blocks.

  • HTML
  • Sources
    • Styles
    • _meta.scss
<div class="meta">
  <span class="meta__item meta__item--type">news article</span>
  <span class="meta__item">17 September 2014</span>
</div>
// Meta

.meta {
  width: 100%;
  line-height: 1.3;
  font-size: $font-size-xsmall;
  letter-spacing: 0.02777778em;
  float: left;
  margin-bottom: 0.9em;

  p {
    margin: 0;
  }
}

.meta__item {
  float: left;
  overflow: hidden;
  white-space: nowrap;
  &:after {
    content: '|';
    color: $grey;
    margin: 0 0.8em;
    position: relative;
    top: -1px;
  }

  &:last-child {
    &:after {
      display: none;
    }
  }
}

.meta__item--type {
  text-transform: uppercase;
}

.meta--header {
  font-size: $font-size-little;
  margin: 0 0 1em;
}

// Drupal fix due to basic_html text formatter settings that are generating
//  tag
.meta {
  .meta__item {
    p {
      font-size: inherit !important;
      display: inline;
      line-height: inherit;
      max-width: none;
    }
  }
}
