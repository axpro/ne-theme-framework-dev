SCSS

SCSS guidelines to a consistent code style

To check (S)CSS files' syntax, we use stylelint with its standard configuration. It is derived from the common rules found within a handful of CSS styleguides, including: The Idiomatic CSS Principles, Github's PrimerCSS Guidelines, Google's CSS Style Guide, Airbnb's Styleguide, and @mdo's Code Guide.
